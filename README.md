# Smart Energy IoT Platform

An IoT + Machine Learning based energy monitoring system that predicts power consumption, detects anomalies, and provides real-time monitoring.

## Features

- IoT sensor simulation
- Power consumption prediction (ML model)
- Anomaly detection
- Decision engine for alerts
- Real-time monitoring dashboard
- Streamlit visualization
- Sensor data logging

## Architecture

Sensor / Simulator
↓
API Server (Flask)
↓
Database (SQLite)
↓
ML Model Prediction
↓
Decision Engine
↓
Streamlit Dashboard
